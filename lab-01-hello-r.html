First lets plot the data in the dino dataset:

dino_data <- datasaurus_dozen %>%
  filter(dataset == "dino")

ggplot(data = dino_data, mapping = aes(x = x, y = y)) +
  geom_point()+
    labs(title = "GOT Scatter Plot")+
    theme_gameofthrones()

The graph looks like a dinosaur. Note if you change the commands to:

ggplot(data = dino_data, mapping = aes(x = y, y = x)) + geom_point()

the dinosaur is lying down!

And next calculate the correlation between x and y in this dataset:

dino_data %>%

Thats a good question. So they should if you give it explicit print options, but otherwise, the defaults provided in R studio and the defaults provided in knitr might not be the same.
